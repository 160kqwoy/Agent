# 主题锚定规范 (Topic Anchoring)

## 1.1 核心主题声明
- **主命题**：Memoria系统——一个具备双重记忆机制的智能助手系统
- **边界范围**：
  - 必须涵盖：项目背景与动机、核心技术实现、测试验证结果、个人学习收获
  - 明确排除：不深入讨论竞品对比、不涉及商业推广、不堆砌技术细节
- **目标读者**：老师、同学（需兼顾技术展示与学习心得分享）

## 1.2 偏题检测机制
生成过程中，每完成一个章节需自检：
- [ ] 本节内容是否直接支撑主命题？
- [ ] 若删除本节，主命题是否仍然完整？（如答案是"是"，则本节可能冗余）
- [ ] 本节是否引入了边界范围外的新话题？

## 1.3 纠偏触发词
当内容偏离时，使用以下句式强制回归：
"回到核心命题：Memoria系统项目汇报，我们需要聚焦于..."
